Kids Safety – API key

Enter the API key from your parent portal. It looks like 53c7fbc2-06be-4a5e-b8d2-43b5e0232efd-agent and sets whether this extension runs in Agent or Control mode.

Suffix -agent = ask-if-real / AI-generated. Suffix -control = blacklist + block.

Change API key

Enter your current API key and a new key from the portal. Both must exist in the portal (backend).

LLM (optional)

Use a local open-source LLM to extract important content from the page before sending to the gateway. Install Ollama and pull a model (e.g. ollama pull llama3.2).

Leave empty to send truncated page text only (no local LLM).
